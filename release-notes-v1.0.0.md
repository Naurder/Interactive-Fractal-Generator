# Interactive Fractal Generator v1.0.0

Initial public portfolio release of the Interactive Fractal Generator.

## Highlights

- Interactive fractal tree generation.
- Live parameter editing through a PyQt5 GUI.
- Recursive depth, branch angle, branch ratio and zoom controls.
- 9-point canvas positioning.
- Antialiasing and left/right branch distinction options.
- Painter customization:
  - brush thickness,
  - line, circle, rectangle and triangle drawing modes,
  - HSV-based color control.
- SVG export for scalable vector output.
- Windows x64 build packaged as a standalone ZIP.

## Download

Use:

```text
Interactive-Fractal-Generator-v1.0.0-windows-x64.zip
```

Unpack the ZIP and run:

```text
main.exe
```

## Notes

The ZIP contains the full PyInstaller build directory. Do not extract only `main.exe`; the bundled dependencies are required.
